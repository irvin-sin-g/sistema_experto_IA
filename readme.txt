Sistema Experto para Selección de Metodologías de Desarrollo
Sistema con factores de certeza que recomienda la metodología de desarrollo ideal según características del proyecto.

Características
10 variables de análisis
10 reglas con factores de certeza (0.65-0.90)
Motor de inferencia forward-chaining
Resultados visuales con % compatibilidad


Uso rápido
bash
git clone https://github.com/irvin-sin-g/sistema_experto_IA
cd sistema-experto-metodologias/src
python sistema_experto.py
